Projekt: To-Do App mit Python

Beschreibung:
Dieses Projekt ist eine einfache To-Do Anwendung, die mit Python entwickelt wurde.
Die Anwendung ermöglich es, Aufgaben hinzuzufügen, anzuzeigen und zu löschen.

Funktionen:
- Aufgaben hinzufügen 
- Aufgaben anzeigen 
- Aufgaben löshen

Technologien:
- Python 
- File Handling
- Unit Testing (unittest)

Meine Role:
Ich habe dieses Projekt Selbstständig entwickelt, um meine Kenntnisse in Python und Software Testing zu verbessern.
